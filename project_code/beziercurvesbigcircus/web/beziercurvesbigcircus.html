This webapp is a DevArt project by Philippe Brouard, updated version 09/2014.

On the main canvas at the top of the display, there is a continuous line drawn with the help of this equation :

x = cos(t) + Aparam*cos(Bparam*t)
y = sint(Cparam*t) + Aparam*sin(Bparam*t)

The drawing is constantly updated. If there is too many lines on the canvas, it's more difficult to update: the application runs slower. Initial frame rate is about 60. It should not go under 6.

When doing nothing on canvas (no keyboard action) during 180 seconds, drawing get back to initial ring.
