BEZIER CURVES BIG CIRCUS

DevArt project by Philippe Brouard, working draft 03/2014...

Laboratory for drawing bezier curves with Dart. The drawing line is calculated with this equation :

x = cos(t) + a*cos(b*t)
y = sint(c*t) + a*sin(b*t)
